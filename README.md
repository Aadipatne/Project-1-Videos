# Project 1 - EDA

The objective of this project is to perform Exploratory Data Analysis (EDA) on a video dataset using Python. The project focuses on understanding the structure of the dataset, checking for missing values and duplicate records, cleaning the data, and exploring relationships between different variables such as views, likes, age, and video category. Various visualizations, including histograms, scatter plots, bar charts, and a correlation heatmap, are used to identify trends and patterns within the data. This project demonstrates the basic steps of data analysis and provides insight into how Python libraries such as Pandas, Matplotlib, and Seaborn can be used to clean, explore, and visualize data.

## Files
- data.csv : Dataset
- eda.py : Python analysis script

## How to Run

1. Download the repository
2. Install Python
3. Run:

python eda.py
